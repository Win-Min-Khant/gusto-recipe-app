# GUSTO Recipe Discovery App

A responsive recipe discovery web application built with React, TypeScript, Tailwind CSS, and React Router. Users can browse recipes, view detailed cooking information, and navigate dynamically between pages.

## Features

- Fetch recipe data from external API
- Dynamic recipe detail pages
- Nested routing with React Router
- Responsive user interface
- Type-safe architecture using TypeScript

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- React Router
- REST API

## Installation

Clone the repository:

```bash
git clone https://github.com/Win-Min-Khant/gusto-recipe-app
```

Navigate to the project folder:

```bash
cd gusto-recipe-app
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
